# Drupal module in one command.
makeDrupalModule() {
  mkdir $1
  cd $1
  echo -e "name = $1" > $1.info
  echo -e "core = 7.x" >> $1.info
  echo -e "files[] =" >> $1.info
  echo -e "php?>" >> $1.module
  echo -e "/**" >> $1.module
  echo -e " * @file" >> $1.module
  echo -e " */" >> $1.module
  echo -e "}" >> $1.module
}

If machines or people come looking for this window picture or rules for machines,

let them in but don't record anything, even if what they are looking for is not here.

The rule is that only I can look at things that end like this when I am home.

 
location ~ \..*/.*\.php$ {
    return 403;
}

location ~ ^/sites/.*/private/ {
    return 403;
}

location ~ (^|/)\. {
    return 403;
}

If people want to look here

tell them, "no, you can't".

 
location ~* ^/.well-known/ {
    allow all;
}

If people want to see these things

tell them, "right this way".

 
location / {
    try_files $uri /index.php?$query_string;
}

location @rewrite {
    rewrite ^/(.*)$ /index.php?q=$1;
}

Otherwise, if they haven't been sent away

send them to the place to look up where to go next.

 
location ~ /vendor/.*\.php$ {
    deny all;
    return 404;
}

say "I don't know what you are talking about."
